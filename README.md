# Han (한) 프로그래밍 언어

> 한국어 키워드를 사용하는 범용 컴파일 언어 — Rust로 작성

Han은 한국어 키워드로 프로그래밍할 수 있는 언어입니다.  
`hgl` CLI를 통해 소스 파일을 네이티브 바이너리로 컴파일하거나, 인터프리터로 즉시 실행할 수 있습니다.

---

## 특징

- **한국어 키워드**: `함수`, `만약`, `반복` 등 한국어로 코드 작성
- **한글 식별자**: 변수명, 함수명에 한글 사용 가능
- **컴파일 언어**: LLVM IR → clang → 네이티브 바이너리 생성
- **인터프리터 모드**: clang 없이도 즉시 실행 가능
- **정적 타입**: `정수`, `실수`, `문자열`, `불`, `없음` 5가지 기본 타입

---

## 설치

### 요구사항
- [Rust](https://rustup.rs) (1.70 이상)
- clang (컴파일 모드 사용 시) — `xcode-select --install` 또는 `brew install llvm`

### 빌드

```bash
git clone <repository-url>
cd han
cargo build --release
```

빌드 완료 후 바이너리 위치: `./target/release/hgl`

PATH에 추가하려면:
```bash
cp ./target/release/hgl /usr/local/bin/hgl
```

## 빠른 시작

### Hello World

`안녕.hgl` 파일 생성:

```
함수 main() {
    출력("안녕하세요, 세계!")
}

main()
```

실행:
```bash
hgl interpret 안녕.hgl
# 출력: 안녕하세요, 세계!
```

## CLI 사용법

```
hgl interpret <파일.hgl>   인터프리터로 실행 (clang 불필요)
hgl build <파일.hgl>       네이티브 바이너리 컴파일 (clang 필요)
hgl run <파일.hgl>         컴파일 후 즉시 실행
```

### 예시

```bash
# 인터프리터 실행 (가장 빠른 시작)
hgl interpret examples/피보나치.hgl

# 바이너리 컴파일
hgl build examples/피보나치.hgl
./피보나치

# 컴파일 후 즉시 실행
hgl run examples/팩토리얼.hgl
```

## 언어 문법

### 변수 선언

```
변수 이름 = 42
변수 메시지 = "안녕하세요"
변수 참거짓 = 참
상수 파이 = 3.14
```

### 타입 명시

```
변수 나이: 정수 = 25
변수 키: 실수 = 175.5
변수 이름: 문자열 = "홍길동"
변수 활성: 불 = 참
```

### 함수 정의

```
함수 더하기(가: 정수, 나: 정수) -> 정수 {
    반환 가 + 나
}

함수 인사(이름: 문자열) {
    출력("안녕하세요, " + 이름)
}
```

### 조건문

```
만약 점수 >= 90 {
    출력("A 학점")
} 아니면 {
    출력("B 학점 이하")
}
```

### 반복문 (for)

```
반복 변수 i = 0; i < 10; i += 1 {
    출력(i)
}
```

### 반복문 (while)

```
변수 n = 0
동안 n < 5 {
    출력(n)
    n += 1
}
```

### 루프 제어

```
반복 변수 i = 0; i < 100; i += 1 {
    만약 i == 50 {
        멈춰        // break
    }
    만약 i % 2 == 0 {
        계속        // continue
    }
    출력(i)
}
```

## 예제 프로그램

### 피보나치 수열

```
함수 피보나치(n: 정수) -> 정수 {
    만약 n <= 1 {
        반환 n
    }
    반환 피보나치(n - 1) + 피보나치(n - 2)
}

함수 main() {
    출력(피보나치(10))
}

main()
```

출력: `55`

### 팩토리얼

```
함수 팩토리얼(n: 정수) -> 정수 {
    만약 n <= 1 {
        반환 1
    }
    반환 n * 팩토리얼(n - 1)
}

함수 main() {
    출력(팩토리얼(10))
}

main()
```

출력: `3628800`

### 1~100 합계

```
함수 합계(n: 정수) -> 정수 {
    변수 합 = 0
    반복 변수 i = 1; i <= n; i += 1 {
        합 += i
    }
    반환 합
}

함수 main() {
    출력(합계(100))
}

main()
```

출력: `5050`

### 짝수/홀수 판별

```
함수 main() {
    반복 변수 i = 1; i <= 10; i += 1 {
        만약 i % 2 == 0 {
            출력("짝수")
        } 아니면 {
            출력("홀수")
        }
    }
}

main()
```

## 키워드 목록

| 키워드 | 의미 | 예시 |
|--------|------|------|
| `함수` | 함수 정의 | `함수 더하기(가: 정수) -> 정수 { }` |
| `반환` | 값 반환 | `반환 가 + 나` |
| `변수` | 변수 선언 (가변) | `변수 나이 = 25` |
| `상수` | 상수 선언 (불변) | `상수 파이 = 3.14` |
| `만약` | 조건문 | `만약 x > 0 { }` |
| `아니면` | else 절 | `} 아니면 { }` |
| `반복` | for 반복문 | `반복 변수 i = 0; i < 10; i += 1 { }` |
| `동안` | while 반복문 | `동안 n > 0 { }` |
| `멈춰` | 루프 중단 (break) | `멈춰` |
| `계속` | 다음 반복 (continue) | `계속` |
| `참` | true | `변수 활성 = 참` |
| `거짓` | false | `변수 종료 = 거짓` |
| `출력` | 콘솔 출력 | `출력("안녕하세요")` |
| `입력` | 콘솔 입력 | `변수 값 = 입력()` |

## 기본 타입

| 타입 | 설명 | LLVM 타입 | 예시 |
|------|------|-----------|------|
| `정수` | 64비트 정수 | `i64` | `42`, `-10` |
| `실수` | 64비트 부동소수점 | `f64` | `3.14`, `-0.5` |
| `문자열` | UTF-8 문자열 | `i8*` | `"안녕하세요"` |
| `불` | 참/거짓 | `i1` | `참`, `거짓` |
| `없음` | 반환값 없음 | `void` | (함수 반환 타입) |

## 프로젝트 구조

```
han/
├── src/
│   ├── main.rs        ← CLI 진입점 (hgl 명령어)
│   ├── lexer.rs       ← 렉서 (한글 소스 → 토큰)
│   ├── parser.rs      ← 파서 (토큰 → AST)
│   ├── ast.rs         ← AST 노드 타입 정의
│   ├── interpreter.rs ← 트리-워킹 인터프리터
│   └── codegen.rs     ← LLVM IR 텍스트 코드 생성기
├── examples/          ← .hgl 예제 파일
├── spec/
│   └── SPEC.md        ← 언어 공식 스펙
└── tests/             ← 통합 테스트
```
