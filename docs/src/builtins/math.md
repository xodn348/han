# Math Functions

| Function | Description | Example | Result |
|----------|-------------|---------|--------|
| `제곱근(x)` | Square root | `제곱근(16.0)` | `4.0` |
| `절댓값(x)` | Absolute value | `절댓값(-5)` | `5` |
| `거듭제곱(밑, 지수)` | Power | `거듭제곱(2, 10)` | `1024.0` |

All math functions accept both `정수` and `실수` inputs.
