# Variables & Constants

## Mutable Variables

```
변수 이름 = 42
변수 메시지 = "안녕하세요"
이름 = 100    // reassignment allowed
```

## Constants

```
상수 파이 = 3.14
// 파이 = 3.0  → runtime error: cannot reassign constant
```

## Type Annotations

```
변수 나이: 정수 = 25
변수 키: 실수 = 175.5
변수 이름: 문자열 = "홍길동"
변수 활성: 불 = 참
```

## Null Values

```
변수 값 = 없음
출력(값)    // 없음
```
